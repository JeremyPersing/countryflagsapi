Country Flags API

Get a country's flag via the country's name, UN Code, ISO Alpha-2 code, or ISO Alpha-3 code. These codes can all be found at iban.com. This API was designed to save developers time from having to download hundreds of flags and handling different cases for identifying how to get a certain country's flag (by name or by code). The flags are also free to use in the public domain.

How to use

We only have png images at 320px and svgs currently. We can get different sizes and variations of flags if needed. It also should be noted that inputs are not case sensitive.

Endpoint
https://countryflagsapi.com/:filetype/:code

:filetype: png or svg

:code: refer to the cards below

Examples
https://countryflagsapi.com/png/br
https://countryflagsapi.com/png/brazil
https://countryflagsapi.com/svg/076
https://countryflagsapi.com/svg/bra
